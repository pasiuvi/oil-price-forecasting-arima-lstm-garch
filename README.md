# Brent Crude Oil Price Forecasting & Volatility Modelling

MSc Data Science coursework (NIB 7072 — Machine Learning, Question 2) —
comparing traditional statistical, machine learning, and deep learning
approaches to forecasting Brent Crude Oil prices, plus volatility modelling
using GARCH.

## Why oil prices

Brent Crude is one of the two global oil price benchmarks and feeds directly
into freight and fuel cost planning in logistics and supply chain operations.
Unlike a generic stock index, this dataset combines a clear long-term trend,
irregular event-driven swings, and strong volatility clustering — making it
a genuinely useful test case for comparing forecasting approaches.

## Project structure

```
├── src/                        Scripts, run in numeric order
│   ├── 01_download_data.py     Pulls Brent Crude data from Yahoo Finance
│   ├── 02_eda_preprocessing.py EDA, stationarity testing, feature engineering
│   ├── 03_model_development.py ARIMA, LSTM, Random Forest — trained & compared
│   ├── 04_garch_volatility.py  GARCH(1,1) / GJR-GARCH volatility modelling
│   └── 05_final_comparison_summary.py   Pulls results together for reporting
├── dashboard/
│   └── app.py                  Interactive Streamlit dashboard
├── data/                        Raw and processed data (generated by scripts)
├── outputs/                     Plots and result CSVs (generated by scripts)
├── requirements.txt
└── INSTRUCTIONS.md              Full setup and run guide
```

## Models compared

| Type | Model |
|---|---|
| Statistical | ARIMA |
| Deep learning | LSTM |
| Machine learning ensemble | Random Forest |
| Volatility | GARCH(1,1), GJR-GARCH(1,1) |

## Quick start

See [INSTRUCTIONS.md](INSTRUCTIONS.md) for the full step-by-step guide,
including Python setup, running the pipeline, and deploying the dashboard.

```bash
pip install -r requirements.txt
cd src
python 01_download_data.py
python 02_eda_preprocessing.py
python 03_model_development.py
python 04_garch_volatility.py
python 05_final_comparison_summary.py
cd ../dashboard
streamlit run app.py
```
